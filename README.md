Logiciel de Gestion d’Agence Immobilière – Java Swing

Logiciel complet de gestion immobilière développé en Java Swing avec une base de données MySQL.
Il permet de gérer les biens, clients, agents, transactions, contrats et rendez-vous via une interface moderne.

- Fonctionnalités principales
- Gestion des biens (ajout, modification, suppression)
- Gestion des clients
- Gestion des agents
- Gestion des transactions
- Gestion des contrats
- Gestion des rendez-vous
- Recherche avancée + interface optimisée
- Architecture MVC
